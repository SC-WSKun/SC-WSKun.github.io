如何使用校园网端口连上自己的服务器

2021-07-16 690 words

购买服务器

  1. CPU这些配置按最低的来，比如1CPU，2G内存，云盘用高效云盘比较便宜
  2. 操作系统可以选windows Server 2019
  3. 网络方面的话华为云可以选择按量付费带宽拉满，可以到300Mbps
  4. 配置安全组：入方向开放TCP3389端口（一般默认打开，用来远程登录），tcp和udp入方向开放53端口或者446端口（这个比较重要）
  5. Win+R调出运行，然后输入mstsc，弹出远程桌面连接，输入你刚刚购买的服务器的公网IP：端口号（端口号只要是入方向开通的都可以），就可以直接进入服务器了

    配置服务器

  6. 下载Xray-Core
    https://github.com/XTLS/Xray-core/releases
  7. 新建txt，输入下面的配置，然后把名字改为config.json，放在解压后的Xray.exe同目录下
    {
      "inbounds": [
        {
          "port": 53,
          "protocol": "vmess",
          "settings": {
            "clients": [
              {
                "id": "814a603c-6b89-4eab-b4ad-d1d5a5403ccd",
                "alterId": 0
              }
            ]
          }
        }
      ],
      "outbounds": [
        {
          "protocol": "freedom"
        }
      ]
    }
  8. 运行Xray.exe，会自动读取config.json文件，注意点开后应该是一直运行的，如果出现闪退可能要以管理员权限打开

    配置客户端

  9. 下载v2ray，不带core的版本
    https://github.com/2dust/v2rayN/releases
  10. 然后把之前下载的Xray-Core文件夹里的除了LICENSE和README之外的三个文件复制到刚刚下载后解压的v2ray文件夹里（相当于用这两个core实现链接服务端和客户端）
  11. 打开v2ray，设置-＞参数设置-＞v2rayN设置-＞core类型改为Xray-Core
  12. 服务器-＞添加Vmess服务器：
    地址：服务器公网IP
    端口：53
    用户ID：json里的uuid，可以自己生成，记得匹配就好
    底层传输安全可以用tls，然后
  13. 右键右下角图标-＞系统代理-＞自动配置系统代理，此时百度ip，看到的是你的服务器ip就说明成功了（注意：原来开着的校园网客户端关闭后并没有完全清除，任务管理器中可以看到有一个sangforprotect的深信服图标的进程，那个会影响你的代理，需要杀死那个进程）
  14. 现在只是能上网，但是无法使用qq和微信，所以需要再下载一个netch（v2ray要在后台一直开着）
    https://github.com/NetchX/Netch/releases
  15. 解压后运行netch.exe，添加socks5服务器，地址使用环路地址127.0.0.1，端口10808，完成后模式选择绕过局域网
  16. 点击设置-＞TUN/TAP-＞自定义DNS-＞DNS：119.29.29.29
  17. 点击直连IP，输入你的服务器公网IP-＞添加-＞保存
  18. 重启netch，此时应该是可以了
