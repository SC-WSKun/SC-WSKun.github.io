ES6的Promise回调

2021-08-04 455 字
JavaScript ES6

Promise 回调

ES6 引入了一种新的对象叫 promise 对象，用来处理回调地狱的问题，把嵌套调用变成链式调用。同时提供了两种状态方便传递不同的返回值。

Promise 回调特点

Promise 对象有一个 then( ) 的方法，可以接收函数的回调返回结果。

then 方法接收两个参数：value 和 reason（一般命名写法，名字可以自定义）。

value 是当 promise 为 resolve 状态返回的值，reason 是当 promise 为 reject 状态返回的值。

then 方法返回一个 promise 对象，因此可以用 then 方法实现链式调用。

promise 对象还有一个 catch( ) 的方法，可以不接收 then 的第一个参数（即 resolve 返回值），直接使用第二个参数（reject 返回值）。本质上是一个语法糖，方便只需要用到 reject 状态的代码的编写。

//实例化Promise对象
const p = new promise(function(resolve,reject)){
    const data = "success";
    //resolve(data); 使promise对象状态为resolve，并且传出data
    const err = "error";
    //reject(err); 使promise对象状态为reject，并且传出err
}


//调用then方法接收回调
p.then(
//当promise状态为resolve，调用第一个参数,value接收resolve的data
function(value){
    console.log(value);
},
//当promise状态为reject，调用第二个参数，reason接收reject的err
function(reason){
    console.log(reason);
})


//then方法也可以只写一个参数，但这个参数必须为resolve对应的参数
p.then(value=>{
    console.log(value);
})


//要想只写一个参数，同时这个参数又是reject对应的参数时，要用到catch方法
p.catch(reason=>{
    console.log(reason);
})


//链式调用
p.then(value => {
    const data1 = "data1";
    resolve (data1);
}).then(value => {
    const data2 = "data2";
    resolve([value,data2]);
}).then(value=>{
    const data3 = "data3";
    value.push(data3);
    resolve(value);
}).then(value=>{
    console.log(value);
})
