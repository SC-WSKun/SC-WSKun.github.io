ES6模板字符串

简述

ES6中新增了一种字符串的书写方式，除了使用双引号(“”)，单引号(‘’)，添加了使用反引号书写的方式(``)。

模板字符串特性：

  1. 保留换行
  2. 可以使用$引入变量，不需要进行字符串拼接
const type = "模板字符串"
const temp = `我是一个${type}`
