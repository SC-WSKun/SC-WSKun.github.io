JSON 简述

JSON 全称为”JavaScript Object Notation”，JavaScript 对象标记法。主要用于 JS 对象与文本之间的转换，方便服务器之间的数据传输。比较类似的是 XML 传输。不同的是 XML 的数据带标签，结构比较复杂。

JSON 数据类型

JSON 可以看作是一些键值对组合起来的数据

JSON 键

要求必须是字符串

JSON 值

只能是以下数据类型

  1. string
  2. bool
  3. number
  4. 数组
  5. 对象

JSON 与文本之间的转换

JSON.parse ( )

将 JSON 文本转换为 JS 对象

JSON.stringify ( )

将 JS 对象转换为 JSON 文本
